# babel埋点方案
分成两类更合适。一类是产品相关的数据指标比如转化率、点击率、UV、PV等等；一类是项目质量、性能相关的，这种可以叫做打点。大公司内部都会有自己的性能平台工具。引入之后可以在一些关键的地方打点做数据统计，比如首屏、http请求、js报错等等，对于业务本身自己也可以做一些手动打点，并且对打点上报的数据自定义报警，线上一旦出错就可以实时的收到报警
